## About This Repo
A Simple Laravel 10 CRUD SPA with InertiaJS , Tailwind Css , Vue , And Jetstream in Dark Mode

## Installation

- Clone this repo Or Download Zip
- Composer Update
- npm install
- npm run dev
- Php artisan serve
